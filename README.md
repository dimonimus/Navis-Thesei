## 1. Общая информация о проекте

**Название проекта**: Arduino MP3 Player с защищённым режимом техника  
**Цель**: создание компактного, надёжного и функционального MP3-плеера на базе недорогих компонентов с возможностью ограничения доступа к расширенным настройкам через PIN-код  
**Платформа**: Arduino Uno / Nano / Pro Mini / RP2040 / ESP32 (основная отладка — на Arduino Nano)  
**Основные модули**:
- DFPlayer Mini (воспроизведение MP3)
- OLED-дисплей SH1106 128×64 (I²C)
- 6 тактовых кнопок
- Светодиод индикации состояния

## 4. Защита настроек (PIN-код)

- Вход в режим техника: удержание кнопки **EQ** ≥ 3 секунды
- Процесс ввода PIN-кода (4 цифры):
  - Vol+ / Vol– → выбор цифры (0–9)
  - Play → подтверждение текущей цифры
  - EQ → отмена / возврат назад
- После 3 неверных попыток — блокировка интерфейса на 10 секунд
- PIN-код хранится в файле `config_secure.h` (не должен попадать в репозиторий!)

## 6. Основные реализованные механизмы и улучшения

- неблокирующая обработка кнопок (с раздельным определением коротких и длинных нажатий)
- точный учёт времени воспроизведения (даже при частых паузах)
- минимальное обновление OLED-дисплея (только при реальном изменении данных)
- включён Watchdog-таймер (WDTO_2S) для защиты от зависаний
- принудительная очистка командного буфера DFPlayer перед отправкой новой команды
- визуальная индикация состояния через LED:
  - постоянное горение — воспроизведение
  - медленное мигание — пауза
  - быстрое мигание — ошибка / буферизация

## 8. Установка и запуск

1. Установить библиотеки:
   - DFRobotDFPlayerMini
   - U8glib
   - (SoftwareSerial встроена в Arduino IDE)

2. Проверить / настроить пины в начале скетча (раздел «Пины»)

3. Подготовить microSD-карту:
   - создать папку `mp3`
   - файлы именовать строго `0001.mp3`, `0002.mp3` … `9999.mp3`

4. Скомпилировать и прошить плату

5. Вставить карту → включить питание

Проект может использоваться как демонстрационный стенд по темам:
- обработка кнопок без delay()
- взаимодействие с модулями по UART / I²C
- защита конфиденциальных данных
- оптимизация ресурсов микроконтроллера
- повышение надёжности встраиваемых систем
